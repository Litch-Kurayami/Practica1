Conclusión

En esta práctica, se logró adquirir experiencia en el uso de Git para la gestión de versiones y Visual Studio Code como entorno de desarrollo. Se comprendió la importancia de la creación de ramas para trabajar de manera paralela en un proyecto y la fusión de cambios para integrar el trabajo realizado en diferentes ramas. Este ejercicio resultó valioso para mejorar la comprensión del flujo de trabajo con Git y Visual Studio Code.
